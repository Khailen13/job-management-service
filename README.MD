# Сервис работы с вакансиями

## Описание:
Программа позволяет получить информацию о вакансиях с платформы hh.ru,
сохранить ее в файл и удобно работать с ней: добавлять, фильтровать, удалять.

### Модуль main.py
#### Функция user_interaction 
Функция для взаимодействия с пользователем через консоль.
Для получения списка вакансий пользователю необходимо ввести:
* поисковый запрос для запроса вакансий из hh.ru;
*  количество вакансий N для топ вакансий по зарплате;
* ключевые слова в наименовании или описании
(для дополнительной фильтрации вакансий, получаемых по поисковому запросу);
* диапазон зарплаты.

### Модуль api.py
#### Класс Parser
Абстрактный класс для работы с API сервиса с вакансиями.
#### Класс HeadHunterAPI (дочерний класс Parser)
Класс для работы с платформой hh.ru. Класс позволяет подключаться к API и получать вакансии.

### Модуль vacancy.py
#### Класс Vacancy
Класс для работы с вакансиями. В классе приняты следующие атрибуты:
* название вакансии (name);
* ссылка на вакансию (url)
* зарплата (salary)
* требования (requirement).

При инициализации атрибутов предусмотрена валидация данных.

При валюте зарплаты отличной от рубля выполняется конвертация в рубли (курсы валют принимаются по API ЦБ РФ).

Класс поддерживает методы сравнения вакансий между собой по зарплате
(больше, меньше, больше или равно, меньше или равно).

Предусмотрен метод cast_to_object_list для вывода информации по атрибутам вакансии в виде списка словарей.

### Модуль file_handler.py
#### Класс FileHandler
Абстрактный класс со следующими методами:
* добавления вакансий в файл (add_vacancy);
* получения данных из файла (get_vacancies);
* удаления информации о вакансии (delete_vacancy).

#### Класс JSONFileHandler (дочерний класс FileHandler)
Класс для сохранения информации о вакансиях в JSON-файл.

Кроме методов родительского класса добавлены следующие методы:
* поиск наличия входной вакансии в существующей базе данных (search_vacancy_in_local_database);
* полное удаление файла с вакансиями (remove_database).

### Модуль excel_file_handler.py
#### Класс ExcelFileHandler (дочерний класс FileHandler)
Класс для сохранения информации о вакансиях в Excel-файл.

Структура аналогична JSONFileHandler.

### Модуль utils.py
В модуле находятся вспомогательные функции для функции взаимодействия с пользователем user_interaction.
#### Функция filter_vacancies
Фильтрует список вакансий по условию наличия хотя бы одного из ключевых слов в имени или требовании вакансии.
#### Функция get_vacancies_by_salary
Фильтрует список вакансий по условию соответствия зарплаты входному диапазону.
Если зарплата вакансии указана в виде диапазона, используется среднее арифметическое значение крайних значений диапазона.
#### Функция sort_vacancies
Сортирует список вакансий по убыванию зарплаты (среднего значения для зарплаты в виде диапазона).
#### Функция get_top_vacancies
Возвращает top_n вакансий по зарплате (на основе отсортированного списка функцией sort_vacancies).
#### Функция print_vacancies
Выводит список вакансий на консоль.
